# 🏥 Healthcare Analytics Dashboard — Power BI

A multi-page Power BI dashboard analyzing 55K+ hospital records across 
patient demographics, operations, finance, and medical insights.

---

## 🛠️ Tools Used
- Power BI Desktop

## 💡 Key Insights
- Diabetes is the biggest cost driver at $236.49M in billing
- Billing peaked in 2021–2022 and dropped sharply in 2024
- Admissions are evenly split across elective, emergency, and urgent types
- Older patients (70–80) make up the largest share of admissions
